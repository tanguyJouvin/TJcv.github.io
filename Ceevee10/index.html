Présentation

D’un esprit créatif et curieux, j’ai choisi de me reconvertir professionnellement. Après avoir exercé diverses professions, j’entreprends un nouveau challenge, celui de devenir Développeur Web. Ce changement de cap a été mûri au cours de mes nombreuses recherches et activités extraprofessionelles. J’apprends de manière autodidacte sur divers sites tels que : openclassroom, free code camp, solo learn et codecadamy. Depuis, j'ai effectué une formation diplomante (niveau 3) à la Wild Code School. J'ai perfectionné mes connaissances notamment en développant avec la librairie React js pour créer des fonctionnalités frontend et Node Js avec Express pour gérer le backend et server-side. Pour ce qui est de l'édition et la gestion de bases de données, j'ai utilisé MySQL et MongoDB, 2 technologies différentes. Je continue mon cursus avec une 3ème année à la Web International School en alternance.
