Présentation

Après avoir obtenu mon titre professionnel de niveau 5 (bac +2) Développeur web et web mobile, je poursuis mon cursus à l'ENI pour devenir concepteur développeur d'applications (niveau 6, équivalent bac + 4).
Lors de mon cursus, J'ai appris l'analyse des besoins du client avec la formalisation des exigences à l'aide de diagramme UML. Ensuite, j'ai approfondi le langages PHP avec Symfony et notamment un projet réalisé avec le Framework Symfony. J'ai découvert JAVA avec le développement de deux applications:
- une application en client lourd avec Java SE
- une application Aavec Java EE.
- une application avec le framework Spring et le moteur de template thymeleaf.
Enfin, durant mon stage de fin d'étude chez Benevolt, j'ai réalisé une application de centralisation de donnée avec le Framework Javascript Nuxt js et un serveur développé à l'aide du Framework PHP Laravel.
